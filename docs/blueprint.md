# **App Name**: ExitPass

## Core Features:

- Role-Based Login: Allows users to log in as Student, Staffulty, Security, or Admin, each with distinct permissions.
- Dashboard Overview: Provides a centralized dashboard displaying relevant information based on the user's role.
- Clearance Management: Enables users to manage clearance items, track their status, and submit for review.
- Profile Management: Allows users to view and update their profile information, including contact details and preferences.
- Form Generation: Generate a clearance form for each user

## Style Guidelines:

- Primary color: Deep navy blue (#1a365d) to convey professionalism and trustworthiness.
- Background color: Very light blue (#F0F4F8) to maintain brightness and good contrast
- Accent color: Maroon (#8C1A3A) used sparingly to highlight key interactive elements, in a nod to school colors.
- Headline font: 'Playfair' serif for titles and headings to evoke elegance.
- Body font: 'Inter' sans-serif for body text to ensure readability and a modern feel.
- Lucide icons used throughout the application, ensuring a consistent and clean visual language.
- Card-based layout with rounded corners and subtle shadows to provide a modern and organized appearance.
- Subtle animations, such as fade-ins and transitions, to enhance user experience without being distracting.
